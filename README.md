# Windows 系统信息一键查询工具

一个简单、轻量、无需安装的 Windows 批处理脚本，一键查看电脑完整硬件与系统信息。

## ✨ 功能特性

- ✅ 查看电脑品牌、型号、序列号
- ✅ 查看 Windows 系统版本
- ✅ 查看 CPU 型号、核心数
- ✅ 查看内存总量、可用内存
- ✅ 查看硬盘容量与使用情况
- ✅ 查看显卡信息
- ✅ 查看主板信息
- ✅ 一键导出信息到桌面文本文件
- ✅ 双击运行，无需任何环境
- ✅ 兼容 Windows 7/8/10/11

## 🚀 使用方法

1. 下载仓库中的 `.bat` 脚本
2. **双击直接运行**
3. 信息会自动显示并导出到桌面

## 📄 导出信息包含

- 计算机品牌 / 型号 / 序列号
- 系统版本与架构
- CPU 处理器信息
- 内存信息
- 硬盘信息
- 显卡信息
- 主板信息

## 📁 文件说明
