# Windows 系统信息一键查询工具

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Windows-blue)](https://www.microsoft.com/windows)
[![.NET](https://img.shields.io/badge/.NET-4.5%2B-green)](https://dotnet.microsoft.com/)

一款功能强大的 Windows 系统硬件与软件信息检测工具，支持基础/完整双模式检测，可一键生成详细报告并保存至桌面。采用 C# 编写，纯命令行界面，轻量免安装。

## ✨ 功能特性

- 🔍 **双模式检测** – 支持“基础配置”与“完整配置”两种信息收集模式，按需选择
- 💾 **一键保存报告** – 按 Enter 键即可将完整报告保存到桌面，文件名自动包含计算机名和时间戳
- 📜 **实时逐行输出** – 模拟命令行逐行打印效果，清晰直观
- 🧠 **智能硬件识别** – 涵盖主板、BIOS、网卡、声卡、电池、显示器等详细硬件信息
- 🇨🇳 **品牌中文识别** – 自动将英文厂商名转换为中文品牌（联想、戴尔、华为等）
- 📅 **出厂日期推测** – 基于 BIOS 发布日期参考，并提示官方保修查询链接
- 🏛️ **信创产品判断** – 根据品牌与 CPU 是否国产，自动判断是否为信创产品
- 🎨 **进度动画反馈** – 收集过程中显示动态进度条和当前检测模块，体验友好

## 📋 功能模块

### 基础模式（选项 1）
- 操作系统信息
- 计算机品牌与型号（含序列号、UUID）
- 处理器信息（核心数、逻辑处理器、频率）
- 内存信息（各插槽容量、总容量）
- 磁盘信息（物理磁盘 + 逻辑分区详情）
- 显卡信息

### 完整模式（选项 2）
包含基础模式所有模块，额外增加：
- 主板信息（制造商、产品、版本、序列号）
- BIOS 信息（名称、版本、发布日期）
- 网卡信息（描述、MAC、IP、子网掩码、网关）
- 声卡信息
- 电池信息（名称、预估剩余时间、状态）
- 显示器信息（名称、分辨率）

### 通用附加功能（两种模式均包含）
- 制造商中文名翻译
- 出厂日期参考检测（BIOS 发布日期）
- 信创产品智能判断（品牌 + CPU）
- 官方售后查询链接建议

  
## 🚀 使用方法

### 运行要求
- Windows 7 / 8 / 10 / 11
- 需要 .NET Framework 4.5 或更高版本（Windows 10/11 通常已内置）
- 建议以管理员身份运行，以获得完整硬件信息（非强制）

### 快速开始
1. 从 [Releases](https://github.com/wzb83889/SystemInfo/releases) 页面下载最新版 `SystemInfo.exe`
2. 双击运行（或右键“以管理员身份运行”）
3. 根据菜单提示输入数字选择模式：
   - `1` – 基础配置检测
   - `2` – 完整配置检测
   - `0` – 退出程序
4. 等待进度条走完，报告将逐行显示在屏幕上
5. 报告显示完毕后：
   - 按 **Enter** 键保存报告到桌面（文件名格式：`系统检测报告_计算机名_时间戳.txt`）
   - 按 **Esc** 键返回主菜单
6. 可重复检测不同模式或退出程序
